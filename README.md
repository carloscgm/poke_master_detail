# poke_master_detail

Example pokemon app with MVVM pattern 

## Some points to note

- MVVM Pattern with injecting dependencies.
- Memory cache in network calls.
- GoRouter.
- Locations.
- Light and Dark Mode.
- Local DDBB with sqlite (favorite pokemon).
- Dio interceptor.
